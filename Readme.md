# Trader Behavior vs Bitcoin Market Sentiment Analysis

This project analyzes how trader behavior (profitability, leverage, volume, and risk)
aligns or diverges from Bitcoin market sentiment classified as Fear or Greed.

## Objective
To uncover patterns between trader actions and market sentiment that can help
inform smarter trading strategies.

## Datasets Used
- Bitcoin Market Sentiment Dataset (Fear & Greed Index)
- Historical Trader Data from Hyperliquid
